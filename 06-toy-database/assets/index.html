B+ Tree 结构

              ┌─────────────────┐
              │   Root (Inner)  │
              │ keys: [12, 25]  │
              └───┬───────┬─────┘
         ┌────────┘       └──────────┐
         ▼                           ▼
┌─────────────────┐         ┌─────────────────┐
│ Inner Node      │         │ Inner Node      │
│ keys: [3, 7]    │         │ keys: [18]      │
└───┬───┬───┐     │         └───┬───────┐     │
    ▼   ▼   ▼     ▼             ▼       ▼     ▼
┌───┐┌───┐┌───┐┌───┐      ┌───┐  ┌───┐
│ L ││ L ││ L ││ L │      │ L │  │ L │
│1 2││3 5││7 9││12 │      │15 │  │18 │
│   ││   ││   ││14 │      │17 │  │25 │
└───┘└───┘└───┘└───┘      └───┘  └───┘
  Leaf nodes linked → for range scans

并发控制 — 锁耦合协议

读操作: 根→叶 共享锁, 获取子节点后释放父节点锁
写操作: 乐观共享锁下行, 叶节点升级为排他锁
         若叶节点需分裂 → 从根重启, 沿途不安全节点加排他锁

快速开始

docker build -t toy-database .
docker run --rm -it -v $(pwd):/workspace toy-database
# 容器内: cargo new toy-db && cd toy-db
