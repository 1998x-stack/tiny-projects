74LS TTLSAP-1 架构 EEPROM 微码~$250 USD

模块分解

Clock

555 Timer ×4, 可调速度, 单步/自动

A & B Registers

74LS173 8位寄存器 ×2, 三态输出

ALU

74LS283 加法器, XOR 减法, 进位标志

RAM

74189 16字节 SRAM, DIP编程

Program Counter

74LS161 4位计数器, 16步

Control Logic

28C16 EEPROM ×3, 微码查找表

Output

74LS273 + 4位数码管显示

Flags

进位(Carry) + 零(Zero) 标志

指令集

指令操作码描述
NOP0000空操作
LDA addr0001从RAM加载到A寄存器
ADD addr0010A = A + RAM[addr]
SUB addr0011A = A - RAM[addr]
STA addr0100存储A到RAM
LDI val0101立即数加载
JMP addr0110无条件跳转
JC addr0111进位则跳转
JZ addr1000为零则跳转
OUT1110输出A到显示
HLT1111停机
