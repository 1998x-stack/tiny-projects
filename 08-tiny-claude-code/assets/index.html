核心代理循环 (永不改变)

User → messages[] → LLM → response
              ↑              │
              │   stop_reason == "tool_use"?
              │      /              \
              │    YES               NO → return text
              │     │
              │     ▼
              └─ execute_tools() → append results

这个 while 循环是整个架构的基础 — 所有功能 (工具、子代理、权限、流式输出) 都在此之上叠加，循环本身永远不变。
