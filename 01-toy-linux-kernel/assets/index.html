功能特性

  • ✓ 引导加载 — BIOS → boot sector → 保护模式 → 内核入口
  • ✓ 多任务调度 — 进程表、上下文切换、轮转调度器
  • ✓ 内存管理 — 物理页分配器、虚拟内存、页表 (两级)
  • ✓ 文件系统 — 缓冲缓存、inode、目录、路径查找、7层设计
  • ✓ 系统调用 — fork/exec/wait/exit/read/write/open/close 等 21 个
  • ✓ 同步机制 — 自旋锁、睡眠锁、sleep/wakeup
  • ✓ 用户空间 — init 进程、shell、基础用户程序

快速开始 (Docker)

  1. 克隆项目：git clone https://github.com/1998x-stack/tiny-projects.git
  2. 进入目录：cd 01-toy-linux-kernel
  3. 构建镜像：docker build -t toy-kernel .
  4. 运行容器：docker run --rm -it toy-kernel
  5. 在容器内：git clone https://github.com/mit-pdos/xv6-riscv.git && cd xv6-riscv && make qemu
