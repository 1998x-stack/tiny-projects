◈ Toy Browser Engine

从零构建的玩具浏览器渲染引擎

Rust / C++ / TypeScript HTML Parser CSS Engine Layout

渲染管线

HTML Tokenizer → DOM Tree
CSS Parser → Rules
Style Cascade → Style Tree
Layout Box Model → Positions
Paint Render → PNG

快速开始

docker build -t toy-browser .
docker run --rm -it -v $(pwd):/workspace toy-browser
# 容器内: cargo new my-browser && cd my-browser
